# 🎬 Video Speed Controller Extension

A lightweight Chrome and Edge extension that lets you change the playback speed of any video on any webpage — up to 10x — with a clean user interface.

---

## 🚀 Features

- ✅ Works on all websites with HTML5 videos (YouTube, Udemy, Coursera, etc.)
- 🔄 Change video speed between **0.1x to 10x**
- ⚡ Instant response with no page reload
- 🌐 Content script handles dynamically loaded videos
- ✅ Save preferred speed in **local storage** (applied automatically on reload).
- ✅ Keyboard shortcuts for quick control:
  - **Shift + {** → Increase speed by **0.25x**
  - **Shift + }** → Decrease speed by **0.25x**

---


---

## 🧩 How to Install (Chrome / Edge)

1. **Download or clone** this repository to your computer.

2. Open your browser and go to: 
chrome://extensions
or
edge://extensions

3. Enable **Developer mode** (top right corner).

4. Click **"Load unpacked"**.

5. Select the `video-speed-extension` folder.

✅ Done! You'll now see the **Video Speed Controller** icon in your toolbar.

---

## 📖 How to Use

1. Visit any website with a video (e.g., YouTube).
2. Click the **Video Speed Controller** icon.
3. Enter a speed between **0.1 and 10**.
4. Click **Set Speed** — your video will instantly update its playback rate.
